## P0 — Foundations (before Week 1) 🔴

Nothing else works until these are done. ~1 week of setup.

- [ ] **Claim handles** `@zidrun` / `@zidrun.dz` on IG, FB Page, TikTok (+ YouTube handle for later). Same avatar (Z mark) + bio everywhere. *(fast, do today)*
- [ ] **Decide the coach persona** — pick the *Coach Zid* route (real ambassador vs AI avatar vs hybrid). See [`04-avatar-and-tools.md`](04-avatar-and-tools.md) §Decision. **Blocks all coach video.**
- [ ] **Lock the coach name + catchphrase** — recommend **"Coach Zid"** + *"Zid!"* (زيد). Get the Darija pun right with 2–3 native runners before it's public.
- [ ] **Link-in-bio landing, coach button first** — *Try your AI coach free — 7 days → /account/coach* · *Find a race → /races* · *Publish → /organizer*. (Linktree/Beacons or a `/start` page.)
- [ ] **Lock the trial model** *(dev ask)* — coach trial **7 days** (down from 1 month) + **reverse trial**: after it ends, keep a small free coach allowance (~5 actions/mo) instead of a hard wall. See `00 §10b`. Unblocks the "7 days" CTA everywhere.
- [ ] **Onboard 10–20 organizers** so real races are live at reveal (kills the "empty app" risk). *(longest lead time — start now)*
- [ ] **Set up ManyChat auto-DM** — "comment ZID → DM the coach link" on IG/FB. Turns coach posts into sign-ups.
- [ ] **UTM + analytics** — GA4 on the site, UTM template (see `02` §6), a tracking sheet, Meta Pixel + TikTok Pixel installed for later retargeting.
- [ ] **Coach demo screen-recordings (once, reuse everywhere)** — capture: onboarding/goal set, a live run with pace + splits, a post-run AI feedback card, a profile-matched tip, a chat reply. This is the raw material for half the coach content.
- [ ] **In-app "share my run/plan" hook** *(dev ask)* — a clean share card (route, distance, pace, "planned by Coach Zid") so every runner becomes a marketer. Highest-leverage growth item.

## P1 — Launch sprint (Weeks 1–6) 🟠

- [ ] **Build the reusable template kit in Canva** — race card, weekend roundup, **coach tip card**, carousel cover, quote/stat card, organizer CTA (see `02` §3).
- [ ] **Batch-shoot Weeks 1–2** — sunrise runs in 2–3 cities + b-roll + the coach demo recordings. One shoot → 8–12 clips.
- [ ] **Produce the "7 days with Coach Zid" series** (coach content bank angle #1) — the flagship coach content; schedule across Week 4 + evergreen.
- [ ] **Set up the coach avatar pipeline** — if AI route: HeyGen/Argil avatar + ElevenLabs FR/AR voice; if real: schedule your ambassador. Ship 3 test coach videos and pick the format that lands.
- [ ] **Ship the 6-week calendar** — Tease → Reveal → Discover → **Coach** → UGC → Milestone. Keep ≥1 coach post per 3.
- [ ] **Founding Runner offer live at Reveal** — badge + **extended trial (e.g. 14 days) or discounted first year** for the first 1,000. Add urgency counter in Stories.
- [ ] **Trial-ending lifecycle nudge** *(dev + marketing)* — in-app + push/email "2 days left with Coach Zid," and a subscribe prompt right after a great run / first PB. Ask at peak motivation — this is where trial→paid is won.
- [ ] **Seed value into 5–10 Algerian running FB groups** — value-first (a coach tip, a race roundup), never spam; respect group rules.
- [ ] **Recruit 3–6 micro-influencers** (running/fitness, 5k–50k) — give them a coach plan to follow on camera; product-for-post or rev-share.
- [ ] **Weekly Friday review** — reach, follower growth, ER, link clicks, **sign-ups + coach-activation %**; kill bottom 20%, scale top pillar/format.
- [ ] **Reply to every comment in first 60 min** + turn good DMs into "Ask Coach Zid" videos.

## P2 — Scale (after Week 2, once winners are clear) 🟡

- [ ] **Boost only proven organic posts** (40k–80k DZD/mo) — target running/fitness interests by wilaya; retarget site visitors with the Founding offer.
- [ ] **UGC engine** — launch `#JeCoursAvecZidRun / #نجري_مع_ZidRun`; re-share the best; a monthly "coach transformation" feature.
- [ ] **Coach success stories / case studies** — real before→after with the plan; the highest-trust proof once you have data.
- [ ] **Club partnerships** — co-promote their races as ZidRun Events; offer free management tools; their members = warm coach signups.
- [ ] **Email/push re-engagement** *(dev + marketing)* — nudge signups who set no goal to try the coach; weekly "your plan is ready" push.
- [ ] **Start a WhatsApp/Telegram broadcast** — huge in Algeria for race countdowns + weekly coach tips.
