# ZidRun — Marketing Folder

**Coach-led** launch-awareness social media campaign for **ZidRun** — a personal **AI running coach** built for Algeria, that's also the one place to discover and register for every race. The coach is the hero (habit + moat); races are the hook. Fronted by the **"Coach Zid"** persona (*"Zid!"* = زيد, "keep going").

**Campaign:** Launch Awareness · **Languages:** FR (primary) + AR + EN · **Platforms:** Instagram · Facebook · TikTok
