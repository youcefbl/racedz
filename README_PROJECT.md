# ZidRun Project Brief

This project follows `algeria-races-codex-brief.md`.

ZidRun is the central Algerian platform for discovering, registering for, and managing running races. The MVP should focus on race discovery, registration, organizer event management, admin approvals, and a clean mobile-first experience.

Use the working name `ZidRun`, package name `racedz`, and database name `racedz`.
