# ZidRun Product UI Captures

This folder stores public-facing app screenshots and UI captures used in marketing, landing pages, app previews, and social/demo materials.

## Current Files

- `app-home-light.webp` — homepage/product screenshot for light-mode contexts.
- `app-home-dark.webp` — homepage/product screenshot for dark-mode contexts.

## Storage Rules

- Keep reusable product screenshots here.
- Keep brand logos and marks in `public/brand/`.
- Keep blog images in `public/blog/`.
- Do not place generated user uploads or payment proofs here; those belong behind the upload/storage boundary.
- Prefer compressed `webp` for UI screenshots unless a target platform requires PNG/JPEG.
